# DevPortfolio

A clean, responsive developer portfolio built with PHP and JavaScript.

![PHP](https://img.shields.io/badge/PHP-8.0+-777BB4?style=flat-square&logo=php&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?style=flat-square&logo=javascript&logoColor=black)

## Features

- Single-page portfolio with smooth scroll navigation
- Project showcase with category filtering
- Skills section with proficiency indicators
- Working contact form with PHP backend and validation
- Responsive layout for all devices
- Dark theme with clean typography

## Setup

```bash
git clone https://github.com/hzwnhrth/DevPortfolio.git
cd DevPortfolio
php -S localhost:8000
```

Open `http://localhost:8000` in your browser.
